# Folder Overview

This folder contains utility modules and constants used across scraping operations. These modules provide shared functionality, constants, and processing utilities for web scraping activities.

## Files

- `Constants.js`: Scraping-related constants and configuration
- `ProcessGameModule.js`: Game data processing utilities
- `ScrapeItems.js`: Scraping item definitions and utilities
- `UtilityFunctions.js`: General utility functions for scraping

## Relations

- Parent folder: [../readMe.md](../readMe.md)
- Key dependencies: Scraping operations, data processing

## Dependencies

- Internal: Used by all scraping modules
- External:
  - Winston (logging)
  - Moment (date handling)
