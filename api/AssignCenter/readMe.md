# Folder Overview

This folder contains modules for assigning competitions and game data to associations and clubs. These modules handle the business logic for linking competitions, teams, and game data within the sports management system.

## Files

- `AssignCompetitions.js`: Assigns competitions to associations
- `assignGameData.js`: Assigns game data to teams and competitions
- `AssignTeamsToCompsAndGrades.js`: Links teams to competitions and grade levels

## Relations

- Parent folder: [../readMe.md](../readMe.md)
- Key dependencies: Data processing modules, competition management

## Dependencies

- Internal: Competition and team management modules
- External: Node-fetch for API calls, Winston for logging
