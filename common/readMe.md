# Folder Overview

This folder contains common utility modules and base controllers used across the application. These modules provide shared functionality including base controllers, dependency management, and disposable resource handling.

## Files

- `BaseController.js`: Base controller with common functionality
- `dependencies.js`: Dependency management and injection
- `Disposable.js`: Disposable resource management utilities

- Key dependencies: Used by all application modules
