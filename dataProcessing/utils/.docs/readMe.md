# Folder Overview

This folder contains data processing utility modules including error handling, memory tracking, processor utilities, and query helpers. These modules provide common functionality used across data processing operations.

## Files

- `errorHandler.js`: Error handling utilities for data processing
- `memoryTracker.js`: Memory usage tracking and monitoring
- `ProcessorUtils.js`: General processor utility functions
- `queryHelpers.js`: Database query helper functions
- `parallelUtils.js`: Parallel processing utilities with concurrency control (Strategy 1: Parallel Page Processing)
