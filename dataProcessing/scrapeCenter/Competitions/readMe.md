# Folder Overview

This folder contains competition scraping modules for extracting competition data from web sources. These modules handle the scraping and processing of competition information for associations.

## Files

- `AssociationCompetitionsFetcher.js`: Fetches competition data for associations
- `getCompetitions.js`: Scrapes competition data from web sources

## Relations

- Parent folder: [../readMe.md](../readMe.md)
- Key dependencies: Puppeteer, scraping utilities

## Dependencies

- Internal: Scraping utilities
- External:
  - Puppeteer (browser automation)
  - Winston (logging)
