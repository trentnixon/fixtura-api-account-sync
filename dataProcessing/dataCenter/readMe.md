# Folder Overview

This folder contains data evaluation modules for associations and clubs. These modules handle the evaluation and assessment of data quality, completeness, and consistency for sports associations and club entities.

## Files

- `evaluateAssociations.js`: Evaluates association data quality and completeness
- `evaluateClubs.js`: Evaluates club data quality and completeness
