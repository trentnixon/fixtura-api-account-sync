# Folder Overview

This folder contains competition-specific assignment modules including creation, CRUD operations, handling, updating, and utility functions. These modules manage all aspects of competition data assignment and management.

## Files

- `CompetitionCreator.js`: Creates new competition entities
- `CompetitionCRUD.js`: CRUD operations for competitions
- `CompetitionHandler.js`: Handles competition processing logic
- `CompetitionUpdater.js`: Updates existing competition data
- `CompetitionUtils.js`: Utility functions for competition operations

## Relations

- Parent folder: [../readMe.md](../readMe.md)
- Key dependencies: Competition management, data processing

## Dependencies

- Internal: Data processing utilities
- External:
  - Winston (logging)
  - Node-fetch (API calls)
